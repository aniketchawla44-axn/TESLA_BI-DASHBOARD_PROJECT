# TESLA_BI DASHBOARD_PROJECT
Key Features & Insights
KPI Cards: Tracking Total Production, Total Deliveries, Average Price, Top Selling Model, and Top Performing Region.
Regional Production (Pie Chart): Breakdown of production units across the Middle East, Asia, Europe, and North America.
Delivery Trends (Area Chart): Historical tracking of estimated vehicle deliveries by year.
Model Analysis (Bar Chart): Comparing range capacities and metrics across Tesla models (Model S, Model X, Model Y, Cybertruck, Model 3).
Environmental Impact (Line Chart): Tracking CO2 savings (in tons) over time.
Interactive Slicers: Dropdown filters for Year and Region to drill down into specific data points dynamically.
🛠️ Tools & Technologies Used
Power BI Desktop: Dashboard design, data visualization, and interactive reporting.
DAX (Data Analysis Expressions): Used for calculated measures and KPI aggregations.
Power Query: Data extraction, cleaning, and transformation (ETL).
